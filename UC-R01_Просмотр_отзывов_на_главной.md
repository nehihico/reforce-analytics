# UC-R01: Просмотр отзывов на главной странице

**Версия:** 1.0 | **Дата:** 2026-06-02 | **Статус:** Черновик

| | |
|---|---|
| **Краткое описание** | Пользователь видит блок с отзывами реальных клиентов на главной странице приложения. Отзывы сгруппированы по всем комплектам, отображаются имя клиента, звёздный рейтинг и текстовый комментарий (если оставлен) |
| **Триггер** | Пользователь открывает главную страницу приложения |

| **Предусловия** | |
|---|---|
| | • Есть хотя бы 1 опубликованный отзыв |

| **Участники** | |
|---|---|
| | • Пользователь (авторизованный или нет) |
| | • Система ЛК |

---

## Основной поток

| Шаг | Действие | Ответ системы |
|-----|----------|--------------|
| 1 | Пользователь открывает главную страницу | Система запрашивает опубликованные отзывы по всем комплектам |
| 2 | — | Система отображает блок «Отзывы» с карточками: имя клиента, звёздный рейтинг (1–5), текст комментария (если заполнен), название комплекта |
| 3 | Пользователь листает отзывы | Система подгружает следующие карточки (если отзывов больше, чем показано изначально) |

---

## Альтернативные потоки

| Условие | Поведение системы |
|---------|------------------|
| Нет опубликованных отзывов | Блок «Отзывы» не отображается на главной |
| Ошибка загрузки данных | Блок не отображается, остальная страница работает корректно |

---

## Постусловие

Пользователь ознакомился с отзывами клиентов.

---

## Критерии приёмки

| # | Критерий | Ожидаемый результат |
|---|---------|-------------------|
| AC-01 | На главной есть 1+ опубликованный отзыв | Блок «Отзывы» отображается |
| AC-02 | Нет ни одного опубликованного отзыва | Блок «Отзывы» не отображается |
| AC-03 | Карточка отзыва с комментарием | Отображаются: имя клиента, звёзды, текст комментария, название комплекта |
| AC-04 | Карточка отзыва без комментария | Отображаются: имя клиента, звёзды, название комплекта; поле комментария отсутствует |
| AC-05 | Отзыв со статусом «На модерации» или «Отклонён» | На главной не отображается |
| AC-06 | Ошибка API при загрузке отзывов | Блок не отображается, остальная страница не ломается |
| AC-07 | Листание отзывов | Следующие карточки подгружаются корректно |
