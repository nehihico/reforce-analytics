# UC-R01: Просмотр отзывов на лендинге

**Версия:** 1.1 | **Дата:** 2026-06-03 | **Статус:** Черновик

| | |
|---|---|
| **Краткое описание** | Посетитель лендинга видит блок с отзывами реальных клиентов. Отзывы охватывают все комплекты, в каждой карточке отображаются полное имя клиента, название комплекта, звёздный рейтинг и текстовый комментарий (если оставлен). Блок оформлен в виде карусели |
| **Триггер** | Посетитель открывает лендинг |

| **Предусловия** | |
|---|---|
| | • Есть хотя бы 1 опубликованный отзыв |

| **Участники** | |
|---|---|
| | • Посетитель (авторизован или нет) |
| | • Система ЛК |

---

## Основной поток

| Шаг | Действие | Ответ системы |
|-----|----------|--------------|
| 1 | Посетитель открывает лендинг | Система загружает опубликованные отзывы по всем комплектам |
| 2 | — | Система отображает блок «Отзывы» в виде карусели. Каждая карточка содержит: полное имя клиента, название комплекта, звёздный рейтинг (1–5), текст комментария (если заполнен) |
| 3 | Посетитель листает карусель | Следующая карточка отзыва становится активной |

---

## Альтернативные потоки

| Условие | Поведение системы |
|---------|------------------|
| Нет опубликованных отзывов | Блок «Отзывы» не отображается на лендинге |
| Ошибка загрузки данных | Блок не отображается, остальная страница работает корректно |

---

## Постусловие

Посетитель ознакомился с отзывами клиентов.

---

## Критерии приёмки

| # | Критерий | Ожидаемый результат |
|---|---------|-------------------|
| AC-01 | На лендинге есть 1+ опубликованный отзыв | Блок «Отзывы» отображается в виде карусели |
| AC-02 | Нет ни одного опубликованного отзыва | Блок «Отзывы» не отображается |
| AC-03 | Карточка отзыва с комментарием | Отображаются: полное имя клиента, название комплекта, звёзды, текст комментария |
| AC-04 | Карточка отзыва без комментария | Отображаются: полное имя клиента, название комплекта, звёзды; поле комментария отсутствует |
| AC-05 | Отзыв со статусом «На модерации» или «Отклонён» | На лендинге не отображается |
| AC-06 | Ошибка API при загрузке отзывов | Блок не отображается, остальная страница не ломается |
| AC-07 | Листание карусели при 2+ отзывах | Карточки переключаются корректно |
| AC-08 | Ровно 1 опубликованный отзыв | Блок отображается как статичная карточка без кнопок навигации и свайпа |
