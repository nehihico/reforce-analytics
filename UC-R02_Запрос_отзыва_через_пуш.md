# UC-R02: Запрос отзыва через пуш-уведомление

**Версия:** 1.0 | **Дата:** 2026-06-02 | **Статус:** Черновик

| | |
|---|---|
| **Краткое описание** | После завершения заказа система отправляет клиенту однократный пуш с предложением оценить комплект. Клиент переходит по пушу, заполняет форму (обязательно — звёзды, опционально — текст до 500 символов) и отправляет отзыв на модерацию |
| **Триггер** | Заказ переведён в статус «Завершён» |

| **Предусловия** | |
|---|---|
| | • Клиент авторизован |
| | • Заказ переведён в статус «Завершён» |
| | • Клиент ещё не оставлял отзыв по данному заказу |
| | • Пуш-уведомления разрешены на устройстве клиента |

| **Участники** | |
|---|---|
| | • Клиент |
| | • Система ЛК |

---

## Основной поток

| Шаг | Действие | Ответ системы |
|-----|----------|--------------|
| 1 | Заказ переведён в статус «Завершён» | Система проверяет: отзыв по заказу не оставлен → отправляет пуш: «Оцените [название комплекта]» |
| 2 | Клиент нажимает на пуш | Система открывает форму-опросник через диплинк |
| 3 | — | Форма отображает: название комплекта, выбор оценки (1–5 звёзд), поле для комментария (опционально, до 500 символов), кнопка «Отправить» |
| 4 | Клиент выбирает оценку | Звёзды подсвечиваются, кнопка «Отправить» становится активной |
| 5 | Клиент вводит комментарий (опционально) | Счётчик символов отображает остаток (например, «120/500») |
| 6 | Клиент нажимает «Отправить» | Система создаёт отзыв со статусом «На модерации». Отображается экран: «Спасибо, ваш отзыв отправлен на проверку» |

---

## Альтернативные потоки

| Условие | Поведение системы |
|---------|------------------|
| Пуш не доставлен / отклонён клиентом | Повторный пуш не отправляется. Отзыв доступен через карточку заказа (UC-R03) |
| Клиент открыл форму, но не отправил | Повторный пуш не отправляется |
| Клиент уже оставил отзыв по этому заказу | Пуш не отправляется |
| Клиент превысил 500 символов | Ввод ограничен: символы сверх лимита не принимаются |
| Клиент нажал «Отправить» без оценки | Кнопка неактивна, отправка невозможна |

---

## Постусловие

Отзыв создан со статусом «На модерации».

---

## Критерии приёмки

| # | Критерий | Ожидаемый результат |
|---|---------|-------------------|
| AC-01 | Заказ завершён, отзыв не оставлен, пуши разрешены | Пуш отправлен однократно |
| AC-02 | Заказ завершён, отзыв уже есть | Пуш не отправляется |
| AC-03 | Заказ завершён, пуши отключены | Пуш не отправляется; отзыв доступен в карточке заказа |
| AC-04 | Клиент не нажал пуш | Повторный пуш не отправляется |
| AC-05 | Клиент нажал пуш | Открывается форма с названием комплекта, звёздами, полем комментария |
| AC-06 | Клиент не выбрал оценку | Кнопка «Отправить» неактивна |
| AC-07 | Клиент выбрал оценку и нажал «Отправить» | Отзыв создан со статусом «На модерации»; экран подтверждения отображается |
| AC-08 | Клиент вводит комментарий | Счётчик символов работает корректно; ввод ограничен 500 символами |
| AC-09 | Клиент отправил форму без комментария | Отзыв создан только с оценкой; поле комментария пустое |
| AC-10 | Отправка формы при нестабильном соединении | Отображается сообщение об ошибке; данные не теряются |
