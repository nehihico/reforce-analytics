# UC-R03: Оставить отзыв из карточки заказа

**Версия:** 1.2 | **Дата:** 2026-06-03 | **Статус:** Черновик

| | |
|---|---|
| **Краткое описание** | Авторизованный клиент открывает карточку завершённого заказа и оставляет отзыв через форму с 3 обязательными оценками (Качество услуги, Работа менеджера, Работа монтажника) и опциональным комментарием. На десктопе форма открывается как модальное окно, на мобайле — как bottom sheet. Форма доступна как запасной путь если клиент не воспользовался пуш-уведомлением (UC-R02) |

> **Скоуп итерации:** в текущей итерации реализуется только в **веб ЛК** (десктоп, модальное окно). На мобайле карточка заказа появится в будущей итерации вместе со встроенным магазином — тогда форма откроется как bottom sheet.
| **Триггер** | Клиент открывает карточку завершённого заказа, по которому отзыв ещё не оставлен |

| **Предусловия** | |
|---|---|
| | • Клиент авторизован |
| | • Заказ имеет статус «Завершён» |
| | • Отзыв по данному заказу ещё не оставлен |

| **Участники** | |
|---|---|
| | • Клиент |
| | • Система ЛК |

---

## Основной поток

| Шаг | Действие | Ответ системы |
|-----|----------|--------------|
| 1 | Клиент открывает карточку завершённого заказа | Система отображает плашку «Оставить отзыв» с названием комплекта |
| 2 | Клиент нажимает на плашку или кнопку «Оставить отзыв» | Открывается форма: десктоп — модальное окно, мобайл — bottom sheet. Форма содержит: название комплекта, три оценки по звёздам — «Качество услуги», «Работа менеджера», «Работа монтажника» (1–5, все обязательны), поле для комментария (опционально, до 500 символов), кнопка «Отправить» (неактивна) |
| 3 | Клиент выставляет все три оценки | Звёзды подсвечиваются. Кнопка «Отправить» становится активной после заполнения всех трёх |
| 4 | Клиент вводит комментарий (опционально) | Счётчик символов отображает остаток |
| 5 | Клиент нажимает «Отправить» | Форма закрывается. Система создаёт отзыв со статусом «На модерации». Плашка заменяется меткой «Отзыв отправлен» |

---

## Альтернативные потоки

| Условие | Поведение системы |
|---------|------------------|
| Отзыв по заказу уже оставлен | Плашка не отображается; вместо неё метка «Отзыв отправлен» |
| Клиент закрыл форму без отправки (✕ или клик на оверлей) | Отзыв не создаётся; плашка остаётся доступной |
| Клиент не заполнил все три оценки | Кнопка «Отправить» неактивна, отправка невозможна |
| Любая из оценок ≤3 звёзд | Поле комментария становится обязательным; метка меняется на «Расскажите, что пошло не так *» |
| Клиент превысил 500 символов | Ввод ограничен: символы сверх лимита не принимаются |
| Ошибка при отправке | Сообщение об ошибке; форма остаётся открытой |

---

## Постусловие

Отзыв создан со статусом «На модерации». Плашка «Оставить отзыв» заменена меткой «Отзыв отправлен».

---

## Критерии приёмки

| # | Критерий | Ожидаемый результат |
|---|---------|-------------------|
| AC-01 | Заказ завершён, отзыв не оставлен | В карточке заказа отображается плашка «Оставить отзыв» с названием комплекта |
| AC-02 | Заказ завершён, отзыв уже оставлен | Плашка не отображается; отображается метка «Отзыв отправлен» |
| AC-03 | Заказ не завершён (любой другой статус) | Плашка «Оставить отзыв» не отображается |
| AC-04 | Клиент нажал на плашку (десктоп) | Открывается модальное окно с формой: название комплекта, три вопроса по звёздам, поле комментария |
| AC-05 | Клиент закрыл форму без отправки (✕ или оверлей) | Отзыв не создан; плашка в карточке остаётся активной |
| AC-06 | Клиент не заполнил все три оценки | Кнопка «Отправить» неактивна |
| AC-07 | Клиент заполнил все три оценки (все ≥4) | Кнопка «Отправить» активна; комментарий опционален |
| AC-08 | Любая из оценок ≤3 | Метка комментария меняется на «Расскажите, что пошло не так *»; кнопка неактивна до заполнения |
| AC-09 | Клиент нажал «Отправить» (форма валидна) | Отзыв создан со статусом «На модерации»; форма закрывается; плашка → метка «Отзыв отправлен» |
| AC-10 | Клиент отправил без комментария (все оценки ≥4) | Отзыв создан; поле комментария пустое |
| AC-11 | Клиент вводит комментарий | Счётчик символов работает корректно; ввод ограничен 500 символами |
| AC-12 | Ошибка при отправке | Сообщение об ошибке; форма не закрывается; данные не теряются |
