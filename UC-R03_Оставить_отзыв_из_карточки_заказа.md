# UC-R03: Оставить отзыв из карточки заказа

**Версия:** 1.0 | **Дата:** 2026-06-02 | **Статус:** Черновик

| | |
|---|---|
| **Краткое описание** | Авторизованный клиент открывает карточку завершённого заказа и оставляет отзыв через bottom sheet. Форма доступна как запасной путь — в случае если клиент не воспользовался пуш-уведомлением (UC-R02) |
| **Триггер** | Клиент открывает карточку завершённого заказа, по которому отзыв ещё не оставлен |

| **Предусловия** | |
|---|---|
| | • Клиент авторизован |
| | • Заказ имеет статус «Завершён» |
| | • Отзыв по данному заказу ещё не оставлен |

| **Участники** | |
|---|---|
| | • Клиент |
| | • Система ЛК |

---

## Основной поток

| Шаг | Действие | Ответ системы |
|-----|----------|--------------|
| 1 | Клиент открывает карточку завершённого заказа | Система отображает кнопку «Оставить отзыв» |
| 2 | Клиент нажимает «Оставить отзыв» | Открывается bottom sheet с формой: название комплекта, выбор оценки (1–5 звёзд), поле для комментария (опционально, до 500 символов), кнопка «Отправить» |
| 3 | Клиент выбирает оценку | Звёзды подсвечиваются, кнопка «Отправить» становится активной |
| 4 | Клиент вводит комментарий (опционально) | Счётчик символов отображает остаток |
| 5 | Клиент нажимает «Отправить» | Система создаёт отзыв со статусом «На модерации». Bottom sheet закрывается. Кнопка «Оставить отзыв» заменяется меткой «Отзыв отправлен» |

---

## Альтернативные потоки

| Условие | Поведение системы |
|---------|------------------|
| Отзыв по заказу уже оставлен | Кнопка «Оставить отзыв» не отображается; вместо неё метка «Отзыв отправлен» |
| Клиент закрыл bottom sheet без отправки | Отзыв не создаётся; кнопка остаётся доступной |
| Клиент нажал «Отправить» без оценки | Кнопка неактивна, отправка невозможна |
| Клиент превысил 500 символов | Ввод ограничен: символы сверх лимита не принимаются |
| Ошибка при отправке | Сообщение об ошибке; bottom sheet остаётся открытым |

---

## Постусловие

Отзыв создан со статусом «На модерации». Кнопка «Оставить отзыв» заменена меткой «Отзыв отправлен».

---

## Критерии приёмки

| # | Критерий | Ожидаемый результат |
|---|---------|-------------------|
| AC-01 | Заказ завершён, отзыв не оставлен | В карточке заказа отображается кнопка «Оставить отзыв» |
| AC-02 | Заказ завершён, отзыв уже оставлен | Кнопка не отображается; отображается метка «Отзыв отправлен» |
| AC-03 | Заказ не завершён (любой другой статус) | Кнопка «Оставить отзыв» не отображается |
| AC-04 | Клиент нажал «Оставить отзыв» | Открывается bottom sheet с формой (название комплекта, звёзды, поле комментария) |
| AC-05 | Клиент закрыл bottom sheet без отправки | Отзыв не создан; кнопка в карточке остаётся активной |
| AC-06 | Клиент не выбрал оценку | Кнопка «Отправить» неактивна |
| AC-07 | Клиент выбрал оценку и нажал «Отправить» | Отзыв создан со статусом «На модерации»; bottom sheet закрывается; появляется метка «Отзыв отправлен» |
| AC-08 | Клиент вводит комментарий | Счётчик символов работает корректно; ввод ограничен 500 символами |
| AC-09 | Клиент отправил без комментария | Отзыв создан только с оценкой |
| AC-10 | Ошибка при отправке | Сообщение об ошибке; bottom sheet не закрывается; данные не теряются |
