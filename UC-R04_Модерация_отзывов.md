# UC-R04: Модерация отзывов в админке

**Версия:** 1.1 | **Дата:** 2026-06-03 | **Статус:** Черновик

| | |
|---|---|
| **Краткое описание** | Администратор просматривает поступившие отзывы и поштучно публикует или отклоняет каждый. Система дополнительно автоматически фильтрует отзывы по стоп-словам до поступления на ручную модерацию. Список стоп-слов администратор ведёт через интерфейс админки |
| **Триггер** | Администратор открывает раздел «Отзывы» в админке |

| **Предусловия** | |
|---|---|
| | • Администратор авторизован в админке |
| | • Есть отзывы со статусом «На модерации» |

| **Участники** | |
|---|---|
| | • Администратор |
| | • Система ЛК |

---

## Основной поток — ручная модерация

| Шаг | Действие | Ответ системы |
|-----|----------|--------------|
| 1 | Администратор открывает раздел «Отзывы» | Система отображает список отзывов с фильтрами: по статусу (На модерации / Опубликован / Отклонён), по комплекту, по дате |
| 2 | Администратор открывает карточку отзыва | Система отображает: имя клиента, название комплекта, оценка (звёзды), текст комментария (если есть), дата создания, кнопки «Опубликовать» и «Отклонить» |
| 3a | Администратор нажимает «Опубликовать» | Статус отзыва меняется на «Опубликован». Отзыв появляется в блоке на главной странице |
| 3b | Администратор нажимает «Отклонить» | Статус меняется на «Отклонён». Клиенту не отображается причина отклонения. Отзыв не публикуется |

---

## Альтернативный поток — автофильтрация по стоп-словам

| Шаг | Действие | Ответ системы |
|-----|----------|--------------|
| 1 | Клиент отправил отзыв с текстом | Система проверяет текст по списку стоп-слов |
| 2a | Стоп-слово не обнаружено | Отзыв поступает на ручную модерацию со статусом «На модерации» |
| 2b | Стоп-слово обнаружено | Отзыв автоматически получает статус «Отклонён» и не поступает на ручную модерацию. Клиенту уведомление не направляется |

---

## Альтернативный поток — управление стоп-словами

| Шаг | Действие | Ответ системы |
|-----|----------|--------------|
| 1 | Администратор открывает раздел управления стоп-словами | Система отображает текущий список стоп-слов |
| 2a | Администратор добавляет новое стоп-слово | Слово добавляется в список; применяется к новым отзывам |
| 2b | Администратор удаляет стоп-слово | Слово удаляется из списка; уже отклонённые отзывы статус не меняют |

---

## Альтернативные потоки

| Условие | Поведение системы |
|---------|------------------|
| Нет отзывов на модерации | Список пуст, отображается заглушка «Нет отзывов на модерации» |
| Ошибка при смене статуса | Сообщение об ошибке; статус отзыва не меняется |
| Отзыв уже имеет статус «Опубликован» или «Отклонён» | Кнопки «Опубликовать» и «Отклонить» недоступны; статус изменить нельзя |

---

## Постусловие

Отзыв опубликован (отображается на главной) или отклонён (не отображается нигде).

---

## Критерии приёмки

| # | Критерий | Ожидаемый результат |
|---|---------|-------------------|
| AC-01 | Открыт раздел «Отзывы», есть отзывы на модерации | Список отображается с фильтрами по статусу, комплекту, дате |
| AC-02 | Нет отзывов ни в одном статусе | Отображается заглушка «Нет отзывов на модерации» |
| AC-03 | Администратор открыл карточку отзыва | Отображаются: имя клиента, комплект, оценка, комментарий, дата, кнопки «Опубликовать» и «Отклонить» |
| AC-04 | Администратор нажал «Опубликовать» | Статус изменился на «Опубликован»; отзыв появился на главной |
| AC-05 | Администратор нажал «Отклонить» | Статус изменился на «Отклонён»; отзыв не отображается нигде в приложении |
| AC-06 | Отклонённый отзыв | Клиент не получает уведомление и не видит причину отклонения |
| AC-07 | Отзыв содержит стоп-слово | Отзыв автоматически получает статус «Отклонён»; на ручную модерацию не поступает |
| AC-08 | Отзыв не содержит стоп-слов | Отзыв поступает на ручную модерацию со статусом «На модерации» |
| AC-09 | Фильтр по статусу «Опубликован» | Список содержит только опубликованные отзывы |
| AC-10 | Ошибка при смене статуса | Сообщение об ошибке; статус отзыва не изменился |
| AC-11 | Администратор добавил новое стоп-слово | Слово появляется в списке; следующий отзыв с этим словом автоматически отклоняется |
| AC-12 | Администратор удалил стоп-слово | Слово убрано из списка; уже отклонённые отзывы остаются со статусом «Отклонён» |
| AC-13 | Отзыв уже опубликован или отклонён | Кнопки «Опубликовать» и «Отклонить» неактивны; повторная смена статуса невозможна |
