# UC-R04: Модерация отзывов в админке

**Версия:** 1.2 | **Дата:** 2026-06-04 | **Статус:** Черновик

| | |
|---|---|
| **Краткое описание** | Администратор просматривает поступившие отзывы и поштучно публикует или отклоняет каждый. Система дополнительно автоматически фильтрует отзывы по стоп-словам до поступления на ручную модерацию. Список стоп-слов администратор ведёт через интерфейс админки |
| **Триггер** | Администратор открывает раздел «Отзывы» в админке |

| **Предусловия** | |
|---|---|
| | • Администратор авторизован в админке |
| | • Есть отзывы со статусом «На модерации» |

| **Участники** | |
|---|---|
| | • Администратор |
| | • Система ЛК |

---

## Основной поток — ручная модерация

| Шаг | Действие | Ответ системы |
|-----|----------|--------------|
| 1 | Администратор открывает раздел «Отзывы» | Система отображает список отзывов с табами-фильтрами по статусу (Все / На модерации / Опубликованные / Отклонённые), дропдаунами по комплекту и дате. На пункте «Отзывы» в навигации отображается бейдж с количеством отзывов «На модерации» |
| 2 | Администратор открывает карточку отзыва (клик по строке) | Система отображает модальное окно: имя клиента, название комплекта, три оценки по звёздам — «Качество услуги», «Работа менеджера», «Работа монтажника», текст комментария (если есть), дата, кнопки «Опубликовать» и «Отклонить» |
| 3a | Администратор нажимает «Опубликовать» | Модал закрывается. Отображается тост: «✓ Отзыв опубликован — он появится на сайте». Статус строки в списке меняется на «Опубликован». Бейдж в навигации обновляется |
| 3b | Администратор нажимает «Отклонить» | Модал закрывается. Отображается тост: «Отзыв отклонён». Статус строки меняется на «Отклонён». Клиенту причина не показывается |

---

## Альтернативный поток — автофильтрация по стоп-словам

| Шаг | Действие | Ответ системы |
|-----|----------|--------------|
| 1 | Клиент отправил отзыв с текстом | Система проверяет текст по списку стоп-слов |
| 2a | Стоп-слово не обнаружено | Отзыв поступает на ручную модерацию со статусом «На модерации» |
| 2b | Стоп-слово обнаружено | Отзыв автоматически получает статус «Отклонён (авто)» и не требует ручной модерации. Отзыв отображается в списке администратора с пометкой «Отклонён (авто)» — доступен для просмотра, но публикация недоступна. Клиенту уведомление не направляется |

---

## Альтернативный поток — управление стоп-словами

| Шаг | Действие | Ответ системы |
|-----|----------|--------------|
| 1 | Администратор открывает раздел управления стоп-словами | Система отображает текущий список стоп-слов |
| 2a | Администратор добавляет новое стоп-слово | Слово добавляется в список; применяется к новым отзывам |
| 2b | Администратор удаляет стоп-слово | Слово удаляется из списка; уже отклонённые отзывы статус не меняют |

---

## Альтернативные потоки

| Условие | Поведение системы |
|---------|------------------|
| Таб «На модерации» пуст | Отображается заглушка «Новых отзывов нет — всё проверено ✓» |
| Таб «Опубликованные» пуст | Отображается заглушка «Опубликованных отзывов пока нет» |
| Таб «Отклонённые» пуст | Отображается заглушка «Отклонённых отзывов нет» |
| Таб «Все» пуст | Отображается заглушка «Отзывов пока нет» |
| Ошибка при смене статуса | Сообщение об ошибке; статус отзыва не меняется |
| Отзыв уже имеет статус «Опубликован» или «Отклонён» | Кнопки «Опубликовать» и «Отклонить» недоступны; статус изменить нельзя |

---

## Постусловие

Отзыв опубликован (отображается на главной) или отклонён (не отображается нигде).

---

## Критерии приёмки

| # | Критерий | Ожидаемый результат |
|---|---------|-------------------|
| AC-01 | Открыт раздел «Отзывы» | Список отображается с табами (Все / На модерации / Опубликованные / Отклонённые), фильтрами по комплекту и дате; бейдж в навигации показывает число отзывов «На модерации» |
| AC-02 | Таб «На модерации» пуст | Отображается заглушка «Новых отзывов нет — всё проверено ✓» |
| AC-03 | Администратор открыл карточку обычного отзыва | Отображаются: имя клиента, комплект, три оценки (Качество услуги / Работа менеджера / Работа монтажника), комментарий, дата, кнопки «Опубликовать» и «Отклонить» |
| AC-04 | Администратор открыл карточку авто-отклонённого отзыва | Кнопок нет; отображается предупреждение «Обнаружено стоп-слово. Ручная публикация недоступна» |
| AC-05 | Администратор нажал «Опубликовать» | Модал закрывается; тост «✓ Отзыв опубликован — он появится на сайте»; статус строки → «Опубликован»; бейдж обновляется |
| AC-06 | Администратор нажал «Отклонить» | Модал закрывается; тост «Отзыв отклонён»; статус строки → «Отклонён» |
| AC-07 | Отклонённый отзыв | Клиент не получает уведомление и не видит причину отклонения |
| AC-08 | Отзыв содержит стоп-слово | Статус «Отклонён (авто)»; отображается в списке; открыть можно, опубликовать — нельзя |
| AC-09 | Отзыв не содержит стоп-слов | Отзыв поступает на ручную модерацию со статусом «На модерации» |
| AC-10 | Фильтр по статусу «Опубликованные» | Список содержит только опубликованные отзывы |
| AC-11 | Отзыв уже опубликован или отклонён вручную | В карточке текст «Отзыв уже [статус] — изменить статус нельзя»; кнопок нет |
| AC-12 | Ошибка при смене статуса | Сообщение об ошибке; статус отзыва не изменился |
| AC-13 | Администратор добавил новое стоп-слово | Слово появляется в списке; следующий отзыв с этим словом автоматически отклоняется |
| AC-14 | Администратор ввёл уже существующее стоп-слово | Поле подсвечивается красным; слово не дублируется |
| AC-15 | Администратор удалил стоп-слово | Слово убрано из списка; уже отклонённые отзывы статус не меняют |
| AC-16 | Список стоп-слов пуст | Отображается заглушка «Список стоп-слов пуст» |
