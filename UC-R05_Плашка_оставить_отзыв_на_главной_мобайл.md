# UC-R05: Плашка «Оставить отзыв» на главной мобильного приложения

**Версия:** 1.1 | **Дата:** 2026-06-03 | **Статус:** Черновик

| | |
|---|---|
| **Краткое описание** | После завершения заказа на главной мобильного приложения появляется плашка с предложением оценить комплект. Плашка остаётся до тех пор, пока клиент не закроет её вручную или не оставит отзыв. Нажатие на плашку открывает bottom sheet с формой-опросником (3 обязательных оценки по звёздам + комментарий; при оценке ≤3 по любому вопросу комментарий становится обязательным) |
| **Триггер** | Клиент открывает главную мобильного приложения, заказ переведён в статус «Завершён», отзыв не оставлен, плашка не закрыта вручную |

| **Предусловия** | |
|---|---|
| | • Клиент авторизован |
| | • Заказ имеет статус «Завершён» |
| | • Отзыв по данному заказу ещё не оставлен |
| | • Клиент не закрывал плашку вручную ранее |

| **Участники** | |
|---|---|
| | • Клиент |
| | • Система ЛК |

> **Вне скоупа итерации:** хранение и отображение отзывов в профиле клиента (раздел «Мои заказы» / «Мои отзывы») запланировано на следующую итерацию. В текущей итерации клиент не видит свои отправленные отзывы внутри приложения.

---

## Основной поток

| Шаг | Действие | Ответ системы |
|-----|----------|--------------|
| 1 | Клиент открывает главную мобильного приложения | Система отображает плашку «Оставить отзыв» под табами (над карточкой объекта): звезда-иконка, название комплекта, кнопка «Оставить отзыв», кнопка закрытия ✕ |
| 2 | Клиент нажимает на плашку или кнопку «Оставить отзыв» | Открывается bottom sheet с формой: название комплекта, три оценки по звёздам — «Качество услуги», «Работа менеджера», «Работа монтажника» (1–5, все обязательны), поле для комментария (опционально, до 500 символов), кнопка «Отправить» (неактивна) |
| 3 | Клиент выставляет все три оценки | Звёзды подсвечиваются. Кнопка «Отправить» становится активной после заполнения всех трёх |
| 4 | Клиент вводит комментарий (опционально) | Счётчик символов отображает остаток |
| 5 | Клиент нажимает «Отправить» | Bottom sheet закрывается. Отзыв создан со статусом «На модерации». Плашка на главной исчезает. Статус модерации клиенту не показывается |

---

## Альтернативные потоки

| Условие | Поведение системы |
|---------|------------------|
| Клиент нажал кнопку закрытия на плашке | Плашка исчезает; отзыв не создаётся. При следующем открытии приложения плашка больше не отображается |
| Клиент свернул приложение без действий | При следующем открытии плашка отображается снова |
| Клиент уже оставил отзыв через пуш (UC-R02) | Плашка не отображается |
| Клиент не заполнил все три оценки | Кнопка «Отправить» неактивна |
| Любая из оценок ≤3 звёзд | Поле комментария становится обязательным; метка меняется на «Расскажите, что пошло не так *»; кнопка неактивна до заполнения комментария |
| Клиент превысил 500 символов | Ввод ограничен: символы сверх лимита не принимаются |
| Ошибка при отправке | Сообщение об ошибке; bottom sheet остаётся открытым, данные не теряются |
| Отзыв отклонён модератором или автофильтром | Клиент не получает уведомление и не видит результат модерации |

---

## Постусловие

Отзыв создан со статусом «На модерации» и плашка скрыта — **или** плашка закрыта клиентом вручную без создания отзыва.

---

## Критерии приёмки

| # | Критерий | Ожидаемый результат |
|---|---------|-------------------|
| AC-01 | Заказ завершён, отзыв не оставлен, плашка не закрывалась | На главной мобильного приложения отображается плашка с названием комплекта |
| AC-02 | Отзыв по заказу уже оставлен (любым способом) | Плашка не отображается |
| AC-03 | Клиент ранее закрыл плашку вручную | Плашка не отображается при повторном открытии приложения |
| AC-04 | Клиент нажал на плашку | Открывается bottom sheet: название комплекта, три вопроса по звёздам («Качество услуги», «Работа менеджера», «Работа монтажника»), поле комментария |
| AC-05 | Клиент нажал кнопку закрытия ✕ на плашке | Плашка исчезает; отзыв не создаётся; повторно не появляется |
| AC-06 | Клиент свернул приложение без действий | При следующем открытии плашка отображается снова |
| AC-07 | Клиент не заполнил все три оценки | Кнопка «Отправить» неактивна |
| AC-08 | Клиент заполнил все три оценки (все ≥4) | Кнопка «Отправить» активна; комментарий опционален |
| AC-09 | Любая из оценок ≤3 | Метка комментария меняется на «Расскажите, что пошло не так *»; кнопка неактивна до заполнения |
| AC-10 | Клиент нажал «Отправить» (форма валидна) | Отзыв создан со статусом «На модерации»; bottom sheet закрывается; плашка исчезает |
| AC-11 | Клиент отправил без комментария (все оценки ≥4) | Отзыв создан; поле комментария пустое |
| AC-12 | Клиент вводит комментарий | Счётчик символов работает корректно; ввод ограничен 500 символами |
| AC-13 | Ошибка при отправке | Сообщение об ошибке; bottom sheet не закрывается; данные не теряются |
| AC-14 | Отзыв отклонён после модерации | Клиент не получает уведомление; статус не отображается |
